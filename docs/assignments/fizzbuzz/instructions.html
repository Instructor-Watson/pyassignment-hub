Goal

Write a function fizzbuzz(n) that returns a list of the numbers from 1 to n (inclusive) with these rules:

  • Multiples of 3 → "Fizz"
  • Multiples of 5 → "Buzz"
  • Multiples of both 3 and 5 → "FizzBuzz"

Steps

  1. Open the template and complete the fizzbuzz function.
  2. Optionally, paste your code in the self‑check area and run public tests.
  3. Submit your final .py file to Gradescope.

Reflection (1–2 sentences)
